## 2. FASE 1 — Seguridad de Código (Signature Flow)

### 2.1 Validación de origen en SignWell postMessage
**Archivo:** `src/pages/SignatureView.tsx`

**Problema:** El handler de `window.addEventListener('message', ...)` no validaba `event.origin`, permitiendo que cualquier sitio web malicioso enviara mensajes que simularan la finalización de firma.

**Solución implementada:**
```typescript
const ALLOWED_SIGNWELL_ORIGINS = [
  'https://www.signwell.com',
  'https://signwell.com',
  'https://app.signwell.com',
];

const handleSignWellMessage = useCallback((event: MessageEvent) => {
  if (!ALLOWED_SIGNWELL_ORIGINS.includes(event.origin)) {
    console.warn('[SignWell] Origen no autorizado:', event.origin);
    return;
  }
  // ... procesar evento
}, [/* deps */]);
```

**Impacto:** Elimina el riesgo de falsificación de firma por sitios de terceros (CWE-345).

### 2.2 Gestión de foco (Focus Management) accesible
**Archivo:** `src/pages/SignatureView.tsx`

**Problema:** Los usuarios de teclado/screen reader no podían navegar eficientemente entre los pasos del flujo de firma (OTP → Consentimiento → Canvas).

**Solución implementada:**
- `otpInputRef`: auto-focus al cargar el paso OTP
- `consentCheckboxRef`: auto-focus al avanzar al paso de consentimiento
- `signatureSectionRef`: scrollIntoView + focus al llegar al canvas de firma
- Transiciones suaves con `behavior: 'smooth'` y `block: 'center'`

### 2.3 Regiones aria-live para anuncios dinámicos
**Archivo:** `src/pages/SignatureView.tsx`

### 6.1 Metodología
Se ejecutaron **12+ agentes en paralelo** organizados en **3 rondas**:

| Ronda | Grupos | Archivos principales |
|-------|--------|---------------------|
| 1 | A-D | Hooks de ventas/firmas/templates, Edge Functions |

### 6.2 Patrones aplicados

#### A. `any` → `unknown`
**Antes:**
```typescript
catch (error: any) {
  toast({ title: 'Error', description: error.message });
}
```

**Después:**
```typescript
catch (error: unknown) {
  const message = error instanceof Error ? error.message : String(error);
  toast({ title: 'Error', description: message });
}
```

**Archivos modificados:** ~75 hooks, ~30 componentes, 22 Edge Functions.

#### B. `as any` → `as unknown as Type`
**Antes:**
```typescript
const sale = data as any;
```

**Después:**
```typescript
const sale = data as unknown as SaleWithRelations;
```

#### C. `Record<string, any>` → `Record<string, unknown>`
**Antes:**
```typescript
const config: Record<string, any> = {};
```

**Después:**
```typescript
const config: Record<string, unknown> = {};
```

#### D. Callbacks tipados
**Antes:**
```typescript
beneficiaries.map((b: any) => b.first_name)
```

**Después:**
```typescript
beneficiaries.map((b: Database['public']['Tables']['beneficiaries']['Row']) => b.first_name)
```
